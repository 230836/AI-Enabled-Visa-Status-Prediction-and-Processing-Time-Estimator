# 🌍 AI-Enabled Visa Status Prediction & Processing Time Estimator

## 📌 Project Overview

This project demonstrates how data-driven approaches can improve transparency and predictability in visa processing systems. It uses machine learning techniques to analyze visa application data and estimate processing time based on user inputs.

## 🎯 Key Features

* 📊 Data preprocessing and feature engineering
* 🤖 Machine learning model training and evaluation
* ⚡ Real-time visa processing time prediction
* 🌐 Interactive and responsive web interface using Streamlit
* 🧪 End-to-end testing with multiple scenarios
